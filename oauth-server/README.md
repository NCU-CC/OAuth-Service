- c3p0 0.9.1.2
- ehcache 2.9.0
- hibernate 4.3.6
- oltu.oauth2 1.0.0
- spring-webmvc 4.1.1
- spring-security 3.2.5

### Gradle
- jettyStart : run embedded server in 'production' environment
- jettyStop  : stop embedded server above
